# NotebookLM for Karim

Clone de l'interface NotebookLM développé en Angular 19 avec Tailwind CSS.

## Prérequis

- Node.js 18+
- npm

## Installation

```bash
npm install
```

## Lancer le projet

```bash
npm start
```

L'application sera disponible sur http://localhost:4200

## Build production

```bash
npm run build
```

## Technologies

- Angular 19 (standalone components)
- Tailwind CSS
- Lucide Angular (icônes)
